# ![Mandarina 🍊](../mandarinalogo.png)

Mandarina is a visual novel engine made for be simple.

## Features

-   📖 Simple verbal language for creating visual novels.
-   🧑‍💻 All in coding, no difficult visual editors.
-   🎨 Fully customizable.
-   👾 Easy to integrate minigames
-   💥 Based in kaboom.js
